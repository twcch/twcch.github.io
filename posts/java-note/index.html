數學運算子 

加: [+] , 減: [-] , 乘: [*] , 除: [/] , 取餘數: [%] , 遞增: [++] , 遞減: [--]

關係運算子 

等於: [==] , 不等於: [!=] , 大於: [>] , 小於: [<] , 大於等於: [>=] , 小於等於: [<=]

邏輯運算子 

1. [&&] AND運算，左右皆為真，結果才為真(true)，否則為假(false)。
2. [||] OR運算，左右皆為假，結果才為假，否則為真。
3. [!] NOT運算，真變假，假變真。

賦值運算子(The Assignment Operators) 

1. [=] 簡單賦值運算子。將右側運算元的值分配給左側運算元。C = A + B將A + B的值分配給C。
2. [+=] 相加與賦值運算子。它將右運算元相加到左運算元並將結果分配給左運算元。C += A等於C = C + A。
3. [-=] 減去與賦值運算子。它從左運算元中減去右運算元，並將結果賦給左運算元。C -= A等於C = C - A。
4. [*=] 乘以與賦值運算子。它將右運算元與左運算元相乘，並將結果賦給左運算元。C *= A等於C = C * A。
5. [/=] 除以與賦值運算子。它將左運算元除以右運算元，並將結果賦給左運算元。C /= A等於C = C / A。
6. [%=] 模數與賦值運算子。它使用兩個運算元來計算獲取模數，並將結果賦給左運算元。C %= A等於C = C % A。
7. [<<=] 左移與賦值運算子。C <<= 2與C = C << 2相同
8. [>>=] 右移與賦值運算子。C >>= 2與C = C >> 2相同
9. [&=] 按位元與賦值運算子。C &= 2與C = C & 2相同
10. [^=] 按位元互斥或和賦值運算子。C ^= 2與C = C ^ 2相同
11. [Ι=] 按位元包含或與賦值運算子。
